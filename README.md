# Booking Application

## Overview
The **Booking Application** is a Java-based system designed to manage reservations for various types of accommodations, including hotels, apartments, villas, and other lodging types. This project was developed as a university assignment to demonstrate **object-oriented programming** in Java.

The application supports both **customers** and **providers/owners** and includes an **administrative interface** for managing users, accommodations, and reservations.

---

## Key Features

### User Management
- **Customer accounts:** registration, login, and profile management.
- **Provider accounts:** hotel or accommodation owners can manage their listings.
- **Admin accounts:** system-wide control over users, accommodations, and reservations.

### Reservation Management
- **Book, view, and cancel reservations.**
- **Check resource availability** before booking.
- **Support for different room types:** single, double, family, maisonette, villa, detached house.

### Accommodation Management
- **Add, update, and remove accommodations.**
- **Support for hotels, apartments, and villas.**
- **Add services to accommodations.**

### Messaging System
- **Send and receive messages** between users and providers.

### Search Functionality
- **Search accommodations and rooms** by various criteria.

### Graphical User Interface
- GUI implemented using Java Swing.
- Dialog-based interaction for customers, providers, and admins.

### Date Management
- **Custom date handling and date picker** for bookings.

### Cancellation Management
- **Make and manage cancellations.**
